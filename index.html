Web 1.0                        1991 - 2003
  • Nacimiento de la World Wide Web (WWW) por Tim Berners-Lee en 1989.
  • Auge de los sitios web informativos y portales.
  • Popularización del correo electrónico y los buscadores
  • Contenido estático y unidireccional.
  • Poca interacción del usuario.
  • Diseño web básico y limitado.
  • Ejemplos: GeoCities, Angelfire, AltaVista.
Web 2.0                        2004 - 2010
  • Explosión de las redes sociales y blogs.
  • Crecimiento del contenido generado por el usuario (UGC).
  • Nacimiento de plataformas web como YouTube, Wikipedia y Facebook.
  • Interactividad y participación del usuario.
  • Contenido dinámico y personalizado.
  • Auge de las aplicaciones web y la Web 2.0.
  • Ejemplos: YouTube, Facebook, Wikipedia, Twitter.
Web 3.0                     2010 - Presente
  • Implementación de la Web semántica y la inteligencia artificial.
  • Mayor interconexión de datos y dispositivos.
  • Nacimiento del blockchain y las criptomonedas.
  • Web descentralizada y abierta.
  • Mayor control del usuario sobre sus datos.
  • Interfaz más natural e inteligente.
  • Ejemplos: Ethereum, Bitcoin, Siri, Google Assistant.
Web 4.0                       En desarrollo
  • Integración de la realidad virtual y aumentada (RV/RA).
  • Nacimiento del metaverso y la Web espacial.
  • Mayor inmersión y personalización del usuario.
  • Interfaz 3D y experiencias inmersivas.
  • Interacción con objetos y avatares virtuales.
  • Mayor inteligencia artificial y aprendizaje automático.
  • Ejemplos: Metaverso de Facebook, Decentraland, VRChat.
